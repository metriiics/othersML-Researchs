# othersML-Researchs

Этот репозиторий содержит исследования по машинному обучению: код экспериментов, датасеты и материалы анализа.


[Credit-Scoring](Credit-Scoring/) - a simple credit score based on personal customer data(простая кредитная оценка по персональным данным клиентов)
	
[HousePredictPrice](HousePredictPrice/) - simple house price prediction(простое предсказание цены дома)

[Titanic](Titanic/) - work on the CRISP-DM methodology with the popular Titanic dataset(работа по методологии CRISP-DM с популярным набором данных "Титаник")
	
[spark-telecom](spark-telecom/) - simple work on apache spark(простая работа по apache spark)
